Paso 1: Detección en Pipeline

OWASP Dependency Check encuentra Log4j vulnerable

info Contexto

Durante la compilación, el servidor de integración continua (Jenkins) ejecuta un escaneo de seguridad automatizado. La herramienta OWASP Dependency Check analiza el archivo pom.xml en busca de bibliotecas con vulnerabilidades conocidas (NVD).

Vulnerabilidad Detectada

Log4Shell (CVE-2021-44228) permite a un atacante ejecutar código arbitrario de forma remota mediante una entrada maliciosa en el sistema de logging.

Impact: Complete system compromise

build_circle

Acción Requerida

El pipeline se detiene automáticamente. El equipo debe actualizar Log4j a la versión 2.17.1 o superior antes de continuar.

Fix: Update dependency in pom.xml
