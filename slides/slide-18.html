Paso 1: Detección en el Pipeline de Compilación

Durante esta fase, el servidor de integración continua (Jenkins) detiene momentáneamente el proceso de compilación para ejecutar un escaneo de seguridad automatizado. La herramienta OWASP Dependency Check analiza el archivo pom.xml en busca de bibliotecas con vulnerabilidades conocidas registradas en la NVD.

jenkins-agent — bash — 80x24

5 [INFO] Checking for updates...
6 [INFO] Analysis Started
