SAST: Static Application Security Testing

Análisis de seguridad estático del código fuente - Caja Blanca

info ¿Qué es SAST?

SAST es una metodología de pruebas de "Caja Blanca" que analiza el código fuente, bytecode o binarios de una aplicación para identificar vulnerabilidades de seguridad.

A diferencia de las pruebas dinámicas, SAST examina la base del código de adentro hacia afuera sin necesidad de ejecutar el programa.

Esto permite a los desarrolladores detectar y remediar fallos de seguridad antes de que la aplicación sea compilada o desplegada.
